# 投資規劃儀表板

安聯大壩 × 0050 資產成長規劃工具

## 功能說明

這是一個純前端互動儀表板，無需後端、無需安裝，直接用瀏覽器開啟即可使用。

### 包含九個分析區塊

| 區塊 | 內容 |
|------|------|
| 01 現況總覽 | 持倉結構、成本基礎、真實報酬說明 |
| 02 策略參數設定 | 滑桿調整投入金額、A 比例、B 期數、目標資產、市場情境 |
| 03 資產情境模擬 | 36 個月折線圖，可切換合計 / 大壩 / 0050 三個視角 |
| 04 B 期數比較 | 6 / 12 / 18 / 24 個月四種期數效益對比，點擊即可切換 |
| 05 市場多空訊號 | 台積電財報、AI 建設週期、融資餘額等六項當前訊號 |
| 06 目標達成進度 | 1 / 2 / 3 年後預估進度條與達標判斷 |
| 07 專屬投資提醒 | 依你的配置與情境動態生成，A 比例 / B 期數 / 情境不同提醒不同 |
| 08 當前配置建議 | 依目前參數即時更新的配置摘要與核心原則 |
| 09 固定策略四情境比較 | A30% B70% 12 月固定策略在牛市 / 先崩後漲 / 橫盤 / 熊市的效益比較，含最佳策略對照表 |

---

## 可互動的參數

- **安聯大壩投入金額**：50 ~ 150 萬（預設 100 萬）
- **A 一次投入比例**：10% ~ 60%（預設 30%）
- **B 定期定額期數**：6 / 12 / 18 / 24 個月（預設 12 個月）
- **目標資產上限**：200 ~ 500 萬（預設 500 萬）
- **市場情境**：牛市繼續 / 先崩後漲 / 橫盤震盪 / 熊市修正（大壩與 0050 同步）

---

## 我的投資背景

- 安聯台灣大壩基金 A 類型：持有 6 年，原始成本約 45 萬，200%+ 報酬後贖回再投入 100 萬
- 0050：現有持倉 180 萬，每月定投 1 萬元
- 固定策略：A=30%（30 萬一次投入）+ B=70%（70 萬分 12 個月，每月 5.8 萬）

## 部署方式

### GitHub Pages（免費，推薦）

1. 建立新的 GitHub Repository（設為 Public）
2. 將 `index.html` 上傳到 Repository 根目錄
3. 進入 `Settings` → `Pages` → Branch 選 `main` → `/ (root)` → 儲存
4. 等待 1~2 分鐘，網址格式為：`https://你的帳號.github.io/你的repo名稱/`

### 本機開啟

直接用瀏覽器開啟 `index.html` 即可，不需要任何伺服器。
